Online utility to schedule round robin tournaments; `pettirosso` is Italian for robin.

## Background

A simple informal application written for the Ulladulla Scrabble Group, but can be used for any pairing.

The initial round is random; scores are not recorded, and subsequent rounds are unweighted so players get a variety of opponents.
